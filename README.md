# PRD Prompt Collection

A curated set of detailed Product Requirements Document (PRD) prompts for building static boilerplate prototypes of common business applications. Each prompt is designed to be used with an AI coding assistant to produce a fully functional, mock-data-driven frontend.

## Purpose

These prompts act as **repeatable, high-signal specifications** — describing layout, data schemas, UI components, interactions, and success criteria — so you can generate a realistic app prototype from a single prompt.

## Template

`PRD - Template.md` — a skeleton to author new PRD prompts. Sections include:

- Core Layout
- Data Structure (TypeScript schemas)
- Implementation Details
- Styling & Theming
- UI Components Specification
- Interactions & States
- Language & Localization
- Example Mock Data
- Success Criteria

## How to Use

1. Pick a PRD file from the list above
2. Copy its contents
3. Paste as a prompt to an AI coding assistant (e.g. opencode, Claude, ChatGPT, etc.)
4. The assistant will generate the full static boilerplate based on the spec
